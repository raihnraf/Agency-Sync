# Patterns from RESEARCH.md

**Rate Limiting Pattern:**
- Configure named rate limiters in bootstrap/app.php using RateLimiter::for()
- Scope by authenticated user: `->by($request->user()?->id ?: $request->ip())`
- Custom response with retry_after: `->response(function () { return response()->json([...], 429); })`
- Apply to routes with `Route::middleware('throttle:limiter-name')`

**Token Expiration Pattern:**
- Custom middleware to check last_used_at timestamp
- Update last_used_at on each request
- Delete token if inactive for 4+ hours
- Return 401 if token expired
- Register middleware alias in bootstrap/app.php

  <action>
Create custom middleware to check and enforce 4-hour token inactivity expiration.

Create app/Http/Middleware/CheckTokenExpiration.php:
```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;
use Laravel\Sanctum\PersonalAccessToken;

class CheckTokenExpiration
{
    public function handle(Request $request, Closure $next)
    {
        $user = $request->user();
        $token = $user?->currentAccessToken();

        if (!$token) {
            return $next($request);
        }

        // Update last_used_at timestamp
        $token->last_used_at = now();
        $token->save();

        // Check if token has been inactive for 4+ hours
        // Use created_at as fallback if last_used_at is null
        $lastActivity = $token->last_used_at ?? $token->created_at;

        if ($lastActivity->lt(now()->subHours(4))) {
            // Delete expired token
            $token->delete();

            return response()->json([
                'errors' => [
                    ['message' => 'Token expired due to inactivity']
                ]
            ], 401);
        }

        return $next($request);
    }
}
```

Per CONTEXT.md decisions:
- "Expire after inactivity — Tokens expire after period of non-use"
- "4-hour duration — Tokens valid for 4 hours during active use"

Per RESEARCH.md pattern: Custom middleware updates last_used_at and checks against 4-hour threshold.
