<task type="auto">
  <name>Task 2: Configure Dusk and install ChromeDriver</name>
  <files>tests/Dusk/dusk.php, .env.dusk.testing, bootstrap/app.php</files>
  <action>
    1. Install Dusk configuration:
       ```bash
       php artisan dusk:install
       ```

    2. Create tests/Dusk/dusk.php configuration:
       ```php
       <?php

       use Laravel\Dusk\Browser;

       return [
           'base_url' => env('DUSK_BASE_URL', 'http://localhost:8000'),
           'chrome_driver' => env('DUSK_CHROME_DRIVER', null),
       ];
       ```

    3. Create .env.dusk.testing file:
       ```
       APP_NAME="AgencySync"
       APP_ENV=testing
       APP_KEY=${APP_KEY}
       APP_DEBUG=true
       APP_URL=http://localhost:8000

       DUSK_ENVIRONMENT=testing
       DUSK_BASE_URL=http://localhost:8000

       DB_CONNECTION=mysql
       DB_HOST=mysql
       DB_PORT=3306
       DB_DATABASE=agencysync_test
       DB_USERNAME=agencysync
       DB_PASSWORD=secret

       BROADCAST_DRIVER=log
       CACHE_DRIVER=file
       FILESYSTEM_DISK=local
       QUEUE_CONNECTION=sync
       SESSION_DRIVER=file
       SESSION_LIFETIME=120
       MATRIX_SCOPED_TENANTS=false
       ```

    4. Update bootstrap/app.php to register Dusk service provider in testing environment:
       ```php
       if ($this->app->environment('testing')) {
           $this->app->register(\Laravel\Dusk\DuskServiceProvider::class);
       }
       ```

    Configuration ensures:
    - Headless Chrome runs in background
    - Separate test database prevents polluting development data
    - Base URL matches Laravel Sail/nginx configuration
  </action>
  <verify>
    <automated>php artisan dusk --version</automated>
  </verify>
  <done>Dusk configured, ChromeDriver installed, testing environment file created</done>
</task>
