       - Fetch products: $products = $service->fetchProducts($tenant, $syncLog)

       - Same chunking and chaining logic

    3. Job chaining pattern:
       ```php
       $chain = collect();
       foreach ($chunks as $chunk) {
           $chain->push(new ProcessProductsChunkJob($syncLog->id, $chunk->toArray()));
       }
       $chain->push(new IndexProductsChunkJob($syncLog->id, $productIds));

       Bus::chain($chain->toArray())->dispatch();
       ```

    Use Laravel Bus::chain() for job chaining.
